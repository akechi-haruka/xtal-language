This is the xtal (Crystal) programming language used for example in Bandai Namco's game "Synchronica" (シンクロニカ).

Some functions do not exist in the game, likewise the game contains some function that are not in this code.

This fork has the error messages translated to English.

Executables:
ix: A line-by-line interactive scripting environment, useful to test syntax and little scripts.
xtal: The main script interpreter executable. Launch with a script filename.

The test directory has some example scripts.
